Có 5 loại token:

- keywords: if else while true false null fn return
- identifiers: chuỗi các chữ số bắt đầu từ 1 chữ cái
- numbers: một dãy số, có thể là số thập phân được ngăn cách bởi dấu .
- string: chuỗi các ký tự nằm trong cặp dấu ""
- symbols: ( ) [ ] { } . , + - \* / % && || ! < > = <= >= == != ; :
